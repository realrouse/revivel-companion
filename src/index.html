First-time setup: Extension ID

To securely connect the ReviveL browser extension, we need its unique Extension ID. This is used for Native Messaging and to restrict access to lbrynet.

  1. Open chrome://extensions (or brave://extensions)
  2. Turn on "Developer mode" (top right)
  3. Find the ReviveL extension
  4. Click the ID (copy it)
  5. Paste below

You can change or reset this later in the Settings tab.
