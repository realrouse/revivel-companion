# ReviveL Companion 0.2.0 — What’s new (plain English)

**Release date:** 2026-07-16

## In one sentence
The Companion can now help the browser extension **send large files** to your computer (for publishing), more safely and reliably than before.

## What this means for you

### Publish bigger content more easily
When you publish videos or other files from the ReviveL browser extension, the Companion can now **receive the file in small pieces**, put them together on your computer, and hand the finished path to the local wallet software. That makes publishing from the browser more practical.

### Still the same simple companion
Everything you already use still works:

- Start / stop the local LBRY wallet software (`lbrynet`)
- Connect the ReviveL extension for full wallet features
- Open `lbry://` links in the ReviveL player
- See status and download/upload statistics

### Safer link between extension and Companion
Communication between the extension and the Companion stays locked to the ReviveL extension only (Native Messaging). Wallet calls can go through this secure path instead of the open web page.

## Who should update
Anyone using ReviveL Companion with the browser extension — especially if you plan to **publish** content.

## How to get it
Download from: https://revivel.app/#downloads  
Or build from this repository at tag/version **0.2.0**.
